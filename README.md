## 📌 Overview
This project is an AI-based Expert System designed to evaluate employee performance using rule-based logic. It simulates human decision-making and provides both evaluation results and improvement suggestions.

---

## 🌐 Live Demo
👉 https://employee-performance-evaluation-system-by-vaishnavi.streamlit.app/

---

## 🧠 Key Features
- 📊 Multi-parameter performance evaluation  
- 💡 Skill-based improvement suggestions  
- 🎯 Decision-level recommendations (Promotion, Appraisal, Training)  
- ⚡ Real-time processing  
- 🎨 Interactive UI using Streamlit  

---

## 📊 Evaluation Parameters
- 🤝 Teamwork  
- ⏰ Punctuality  
- 📌 Task Completion  
- 🗣️ Communication Skills  
- 🧠 Problem Solving  
- 👨‍💼 Leadership  

---

## ⚙️ System Workflow
1. Enter employee details  
2. Provide ratings (1–5 scale)  
3. System calculates average score  
4. Applies rule-based inference  
5. Generates:
   - Performance level  
   - Personalized suggestions  
   - Final recommendation
